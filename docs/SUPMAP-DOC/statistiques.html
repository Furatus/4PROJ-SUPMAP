BarChart

Données

Créées à partir de l’heure de création :

const hoursMap = Array(24).fill(0); data.forEach((incident) => { const hour = new Date(incident.createdAt).getHours(); hoursMap[hour]++; }); const hourlyData = hoursMap.map((count, hour) => ({ hour: `${hour.toString().padStart(2, '0')}h`, count, }));
